# Документы по ТЗ

Эта папка содержит материалы, относящиеся только к исходному техническому заданию на разработку ETL-пайплайна.

В ТЗ зафиксированы следующие задачи:

1. Анализ исходных данных.
2. Выбор модели и подхода.
3. Разработка ETL-пайплайна.
4. Тестирование на данных заказчика.
5. Разработка API.

Файлы в папке:

- `PROJECT_DESCRIPTION_TZ.md` - описание проекта в границах ТЗ.
- `TASK_MAPPING.md` - соответствие задач ТЗ реализованным компонентам и артефактам.

Материалы про расширенный RAG-контур, golden QA, retrieval-метрики и последующие исследовательские эксперименты вынесены за рамки этой папки. Они могут использоваться как развитие проекта, но не являются основной формулировкой результата по ТЗ.
